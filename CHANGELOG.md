# Changelog

All notable changes to the Keyfount extension are documented here. The version
of each release is driven by the `release:*` label on its `develop → main` pull
request; the git tag `vX.Y.Z` is the source of truth.

## v0.0.1 — 2026-05-28

Initial packaged release of the Keyfount browser extension, built for Chrome and Firefox.
